Grocery App - README
Project Overview
This project is a full-stack Grocery Web Application built using React (frontend), Node.js + Express (backend), and MongoDB Atlas (database). The app allows users to view, add, update, and delete grocery products with a clean UI and smooth backend integration.
Project Requirements
- Build a CRUD grocery management system
- Use MongoDB Atlas for cloud storage
- Use Express.js for backend API
- React UI for user‑friendly product management

Wireframes
- Home Page
- Add Product Form
- Product Dashboard

Functional Requirements / Use Cases
1. Add new grocery products
2. View all available products
3. Edit/update product information
4. Delete products
5. Database automatically syncs with backend

Technical Details - Tech Stack
Frontend: React + Vite
Backend: Node.js + Express
Database: MongoDB Atlas
Styling: CSS / Tailwind (optional)
API: RESTful Express routes
